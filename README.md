实验说明

MLP实验结果
网络结构：784-512-256-128-10，使用ReLU激活函数、BatchNorm和Dropout(0.3)正则化 
训练参数：Adam优化器(lr=0.001)，batch_size=128，训练15个epoch  
最终准确率：测试集准确率98.60%，达到≥98%的目标

CNN实验结果  
网络结构：3层卷积(32-64-128通道，第一层5×5卷积核，后续3×3卷积核)，每层接MaxPooling和BatchNorm，全连接层1152-256-128-10，使用Dropout(0.25)正则化
训练参数：Adam优化器(lr=0.001)，batch_size=128，训练15个epoch  
最终准确率：测试集准确率99.39%，达到≥99%的目标

结论
CNN在图像分类任务上明显优于MLP，使用更少参数获得更高准确率，验证了卷积操作在提取图像空间特征方面的优势。
